# 🎓 Programa Rededaora — Polo Cangaíba

Plataforma educacional desenvolvida para disponibilização de materiais didáticos do curso de **Programação e Desenvolvimento Web**, ministrado no Polo Cangaíba.

O projeto foi criado com foco em **UX moderno, responsividade e experiência de aprendizagem simples e acessível**, permitindo que as alunas acessem conteúdos diretamente pelo navegador.

---

## 🚀 Acesso ao projeto

Após publicado no GitHub Pages:

👉 https://vanessabaldin1.github.io/programacao-web-polo-cangaiba

---

## 📚 Funcionalidades

- 🔐 Tela de login com autenticação simples (sem backend)
- 📱 Layout responsivo (mobile-first)
- 📄 Acesso a materiais em PDF
- 🎧 Podcasts em áudio (M4A/MP3)
- 🎵 Músicas de fixação de conteúdo
- 📸 Mural de fotos das aulas
- 🎬 Mural de vídeos das aulas
- 🌙 Modo claro/escuro (dark mode)
- ⬆ Botão de scroll suave para navegação
- 👩‍🏫 Identificação personalizada do usuário logado
- 🚪 Botão de logout
